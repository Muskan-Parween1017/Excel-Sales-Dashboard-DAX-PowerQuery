# 📊 Sales Analytics Dashboard using Excel (Power Query + DAX)

# Project Overview

This project demonstrates advanced data analysis using Microsoft Excel. The dataset was cleaned and transformed using Power Query, and analytical measures were created using DAX to generate business insights.

# Tools & Skills Used

Microsoft Excel

Power Query (ETL Process)

DAX Measures

Data Modeling

Pivot Tables & Slicers

KPI Analysis

## Key Features

Dynamic KPIs (Total Sales, Revenue, Growth %)

Region-wise and Product-wise performance analysis

Time-series trend analysis

Interactive dashboard with slicers

Automated data cleaning using Power Query

## Business Insights

Identified top-performing regions

Sales growth trends over time

Product category profitability comparison
